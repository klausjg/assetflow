## 🔧 변경/개선
- **업데이트 내역 모달**: 각 릴리즈 카드에서 제목 자리에 "이번 릴리즈 요약" 내용이 바로 표시됩니다. 무슨 업데이트인지 펼쳐보지 않아도 바로 파악할 수 있어요.
- **What's New 팝업**: 새 버전이 배포되면 뜨는 안내 팝업도 동일한 방식으로 정리되어, 중복된 제목 없이 요약 문구가 돋보이게 배치됩니다.
- 제목과 본문의 시각적 위계가 명확해져서 릴리즈 노트를 훑어보기 편해졌습니다.

## 🐛 수정된 문제
- 릴리즈 타이틀과 "이번 릴리즈 요약"이 따로 표시되어 내용이 중복돼 보이던 문제를 정리했어요.
