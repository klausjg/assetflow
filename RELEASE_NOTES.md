## 🎉 이번 릴리즈 요약
- 국내 상장 해외 ETF를 **미국 ETF로 정확히 분류**
- 청년도약계좌·청약 등 **개인 자산 등록 가능** (티커=`0`)
- DB 시트 **수식 자동 채움** 기능 추가

## ✨ 추가된 기능
- **사용자 정의 자산**: 티커에 `0` 입력 → 원하는 이름으로 자산 등록. "달러" 키워드 있으면 자동으로 USD 처리.
- **신규 카테고리 자동 등록**: 없는 카테고리 입력 시 "추가할까요?" 팝업. YES 한 번으로 등록+재정렬 완료.
- **수식 자동 주입**: 계좌명·티커 입력만 해도 나머지 열 수식이 자동으로 채워짐.
- **🔄 DB 수식 최신화 메뉴**: 전체 행 수식을 최신 버전으로 일괄 갱신.

## 🔧 변경/개선
- TIGER 미국S&P500, RISE US NASDAQ100 등 → **"미국 ETF"로 재분류** (실제 미국 비중 반영)
- 예수금 코드 `1`/`2` → **`0` 하나로 통합**
- "🛠️ 데이터 강제 재정렬" → **"🔄 자산 구분별 데이터 최신화"**

## 🐛 수정된 문제
- 티커 `0`인 자유입력 종목이 일간 기록에 0원으로 집계되던 오류 수정

## 📝 기타
- 기존 사본은 **새 사본 재생성 권장**. DB 내용 복사 → 새 사본에 값 붙여넣기 → "🔄 DB 수식 최신화" 클릭으로 이관.
